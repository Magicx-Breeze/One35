# Welcome to MgaicX One 35!

**This repository contains instructions for the MagicX One 35 Dawn front-end / factory reset.**


# Dawn Regarding Support for More Platforms

**Since I've streamlined the MagicX One 35 handheld device by removing many platforms, if any player requires complete platform support, they can download the EmulatorConfig.json file from this repository, overwrite the existing file in**
> /Android/data/magicx.touch.one/files/Config/json/
 
**in internal storage, and then refresh Dawn.**
